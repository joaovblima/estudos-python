# 📚 Estudos Python

Repositório dedicado ao meu aprendizado contínuo em **Python**, contendo exercícios resolvidos, anotações e práticas baseadas em livros e materiais de estudo.

## 📌 Sobre o projeto

Este repositório começou com a resolução dos exercícios do livro *Python Crash Course*, e será expandido com exercícios e desafios de outros livros e fontes ao longo da minha jornada.

A ideia é centralizar tudo em um só lugar, facilitando a revisão e acompanhando minha evolução como desenvolvedor.

## 🧠 Conteúdo

- ✔️ Exercícios resolvidos do *Python Crash Course*
- 📖 Exercícios de outros livros (em breve)
- 🧪 Pequenos experimentos e testes
- 📝 Anotações importantes

## 📂 Estrutura

A estrutura do repositório será organizada por fonte de estudo:
``` text 
estudos-python/
│
├── python-crash-course/
│ ├── cap01/
│ ├── cap02/
│ └── ...
│
└── extras/
```
## 🚀 Objetivo

- Praticar Python de forma consistente  
- Consolidar fundamentos da linguagem

- Desenvolver disciplina de estudo  

## 📈 Progresso

Este repositório será atualizado constantemente conforme avanço nos estudos.
